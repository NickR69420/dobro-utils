# Dobro-Utils

Originally made for my bot Dobro but can be used by all!

## Usage

# Javascript

```js
const { Utils } = require("dobro-utils");

const util = new Utils();

util.formatString("HELLO"); // Hello
```

# Typescript

```ts
import { Utils } from "dobro-utils";

const util = new Utils();

util.formatString("HELLO"); // Hello
```

Enjoy!
